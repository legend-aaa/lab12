Простой блог на PHP с использованием JSON для хранения данных

Возможности: 📝 Создание и просмотр постов; 💬 Комментирование постов; 👤 Регистрация и авторизация пользователей; 🖼️ Загрузка изображений к постам; 🎨 Адаптивный дизайн;
